# DataCo-Supply-Chain-Dashboard
Analyzed the DataCo Global Supply Chain dataset using Python, Pandas, and NumPy for cleaning and preprocessing. Integrated structured and unstructured data to explore end-to-end logistics performance. Built interactive Power BI visuals to uncover trends, delays, and operational insights.
